# 📱 QR Code Generator

A simple, modern, and fully offline **QR Code Generator** built with **HTML, CSS (Tailwind)**, and **JavaScript**.  
It allows you to generate customizable QR codes instantly, preview them, and download or copy them with smooth animations and a responsive UI.

---

## ✨ Features

- 🔗 Generate QR codes from **any text or URL**  
- 🎨 Customize:
  - Size (128px – 640px)
  - Foreground and background colors
  - Margin / padding around QR
  - Error correction level (L, M, Q, H)  
- 📲 Download as **PNG**  
- 📋 Copy QR code image directly to clipboard (supported browsers)  
- 💾 Saves your last settings in **localStorage**  
- 🖱️ Drag & Drop text support  
- ⚡ Works completely **offline** (client-side only)  
- 🎭 Animated, responsive, and clean **TailwindCSS** UI  
- 🌓 Dark/Light theme toggle
